Emulsion

interactive audio-reactive fluid simulation with TouchDesigner

Emulsion is an interactive fluid simulation made using custom glsl shaders in TouchDesigner. The user can move the paintbrush around the screen using their hand (detected via MoveNet) while the state of the fluid changes to react to audio playing along with it. The kick from the music refreshes the cycle, setting the user's paint splat as the background and fading out the previous one.

The textured 3D visual is inspired by hardening emulsion but made entirely using 2D shaders and feedback. MoveNet applied using Torin Blankensmith's No-Kinect MoveNet applet found here.
