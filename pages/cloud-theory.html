Cloud Theory

GPT-2 based cloud watching companion
Watch the snowfall as you read about clouds

Documentation Promo Video

Cloud Theory is a relaxing videogame where you lay on the ground and look at clouds. Every once in a while you or your friend might see a shape in the clouds, which sparks conversation about everything from the origin of poodles to the deepest mysteries of life. Launching sometime in 2023.

Cloud Theory is a project output of Colleen Macklin's research project called Machine Unlearning. Cloud Theory is an experiemental fine-tuned a GPT-based dialogue engine focused on asking existential questions that can be accessed live in a game format. I programmed parts of this game in Unity including work on text analysis, live GPT-2 x Unity integration and text-to-speech programming.
