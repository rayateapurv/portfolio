Yumeboshi

interactive music visualizer using shaders in three.js
Watch the generative paint drip across the page

Try It! Process

Yumeboshi is a set of two 3D lava lamp-like blobs that react to user input (audio). The project consists of a three.js scene setup along with an interactive GUI and basic html that takes in audio input. Mesh distortion is based on audio analysis done using the WebAudio API. Visuals, consist of two separate shaders, one is a combination of noise masks and the other is a zoom blur creating a luminous effect.
