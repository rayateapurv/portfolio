Description

PhotoCrit: Fish Edition is a photography game where you find and photograph critically endangered fish in a digital aquarium. Using a custom-made polaroid camera controller, players can take pictures of and learn more about the fish in the game in real time. The game is a gallery that represents the biodiversity we are losing to progress and uses digital technology to permanently enshrine species that will be lost to time in the next few years.
Data sourced from the IUCN Red List.

Goal
Create a data visualisation of Critically Endangered fish species and present it in a fun interactive format.

Role

Game Developer / Hardware Developer

  • Programmed a complete game in Unity to visualise Critically Endangered fish species newly entered into the IUCN Red List in the last two years.
  • Built a physical controller for the game that mimics the functions of a polaroid camera using Arduino and a Thermal Printer and connected it to the game in Unity.
  • Ensured robustness of the game and controller for a long public exhibition (two months).

Outcomes

  • Visualised critical data in a game format with an alt controller that gave a physical keepsake to its players for playing.
  • The project was presented at Aalto Media Lab Winter Demo Day 2024 and installed at the Marsio Gallery as part of the Aalto Media Lab 30 Years exhibition from Dec24-Jan25.
