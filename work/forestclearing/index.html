Description

A Forest Clearing is a collaborative multiplayer experience built using the p5 party library where players grow and maintain their forest. In this gamified view on deforestation, players are challenged by an increasing number of NPC lumberjacks that cut down the trees they grow. The game challenges traditional resource gathering in games and reverses it, making the player responsible for the game's environment.
The initial prototype of this game was made in a week and won the Parsons DT Game Jam 2022.

Goal
Create a multiplayer experience to create awareness about deforestation.

Role

Full Stack Game Developer

  • Programmed the NPC and tree growth logic.
  • Supported with front-end features like a waiting room and status UI.
