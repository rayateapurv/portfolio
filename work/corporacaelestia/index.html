Description

Corpora Caelestia is a visualisation of satellites in geostationary(GEO) orbit that were launched from 2019-2021. It showcases the inequalities between the world's nations using satellites as an indicator of power and status. To spatially grasp complex satellite data, the visualisation is built as a 3D website using the three.js library.
Data sourced from the UCS Satellite Database.

Goal
Simplify complex data about satellites in an interactive, easy to use application.

Role

App Developer

  • Sorted and Analysed data about satellites in GEO orbit from the UCS Satellite Database.
  • Built a 3D website using three.js that showcases important details about the data in a visual format.
  • Each satellite is procedurally generated based on factors like purpose, year of launch, etc. and mapped to its location in space using actual satellite position data.

Outcomes

  • Learnt data cleaning and analysis using Observable to filter relevant data.
  • Made the app publically accessible by launching it online.
