Description

Tiny Criminals is a fast paced racing game with cozy sim elements made in Godot. In Tiny Criminals, life’s all about vibing on your own terms. Plan a heist, grab some loot, then kick back on your island hideout. No stress, no rules — just sun, schemes, and a whole lot of fun. This isn’t about survival. It’s about enjoying life, one “borrowed” item at a time.

Opportunity
Create a complete game using the open-source game engine, Godot.

Role

Game Developer

  • Prototyped and tested fun game mechanics within Godot.
  • Used GDScript to implement NPC behaviours and animations.
  • Worked on gameplay actions like the in-game shop functions and decoration placement.
  • Worked with people from multiple disciplines to scope, test and ship a complete game product.

Outcomes

  • Learnt to make a game using the Godot engine.
  • Built a robust and fun experience that was shown during Aalto Media Lab Spring Demo Day 2025.
  • Shipped the full game downloadable on itch.io.
