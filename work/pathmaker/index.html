Description

Pathmaker is a browser-based collaborative multiplayer game in which you use your hands to lead an object to its goal. Inspired by Lemmings and playable in the browser, it combines computer vision with physics-based mechanics. The project explores interaction design using AI by replacing traditional inputs (like a mouse or keyboard) with hand gestures, using Mediapipeâ€™s hands library for hand tracking and matter.js as the physics engine to drive the game world.

Goal
Experiment and prototype possibilities of computer vision and voice based AI as an alternative to GUIs.

Role

Design Engineer

  • Researched and prototyped embodied methods of interaction using AI, TouchDesigner and body sensors.
  • Designed and programmed a game concept to suit the interaction.
  • Tested different computer vision models (MoveNet, Mediapipe Pose, Mediapipe Hands, BodyPose, Teachable Machine) to determine one suitable for a multiplayer game.

Outcomes

  • Programmed an interface with mediapipe to use hands as interaction devices.
  • Built a multiplayer game in the browser playable by upto 4 people.
  • The game is shipped and currently playable online.
